# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/lucaslauriano/survivors-challange.git
```

```bash
# Install Dependencies
$ yarn
```

```bash
$ yarn dev
```

Go to http://localhost:3000/survivors to see the result.
